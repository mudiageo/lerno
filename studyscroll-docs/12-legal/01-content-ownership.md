## 2. YouTube API Compliance

StudyScroll embeds YouTube videos using the official YouTube IFrame API. This means:

### What we DO:
- Embed videos using `<iframe>` with `youtube-nocookie.com` domain (privacy-enhanced mode)
- Use the YouTube IFrame Player API for playback control
- Show YouTube branding as required
- Link back to YouTube for full video viewing

### What we DO NOT do:
- Download YouTube videos to our servers (violates YouTube ToS Section 5.1.H)
- Extract YouTube audio streams
- Bypass YouTube's `allowEmbed` restrictions
- Cache YouTube thumbnails on our servers (fetch from `img.youtube.com` directly in browser)
- Use youtube-dl/yt-dlp to scrape content — **this is prohibited**

### YouTube Data API Usage:
```typescript
// Fetching video metadata (requires YouTube Data API v3 key)
// apps/web/src/lib/server/remote/video.ts

export const searchYouTubeVideos = query(async ({ query, courseCode }) => {
  const url = new URL('https://www.googleapis.com/youtube/v3/search');
  url.searchParams.set('key', process.env.YOUTUBE_API_KEY!);
  url.searchParams.set('q', `${query} ${courseCode} lecture tutorial`);
  url.searchParams.set('type', 'video');
  url.searchParams.set('videoEmbeddable', 'true');
  url.searchParams.set('maxResults', '10');
  url.searchParams.set('part', 'snippet');
  
  const res = await fetch(url);
  const data = await res.json();
  return data.items;
});
```

### Privacy-Enhanced Embed:
```svelte
<!-- src/lib/components/video/YouTubeEmbed.svelte -->
<script>
  let { videoId, title } = $props();
  // youtube-nocookie.com = privacy-enhanced mode (no tracking cookies until play)

<iframe

```

### YouTube API Quota
- Free quota: 10,000 units/day
- Search costs 100 units per call
- Implement aggressive caching: store search results in PostgreSQL for 24h
- Cache video metadata for 7 days

```typescript
// Cache YouTube search results to avoid quota exhaustion
export const getCachedYouTubeResults = query(async ({ query, courseCode }) => {
  const cacheKey = `yt:${courseCode}:${query}`;
  const cached = await db.query.youtubeCache.findFirst({
    where: and(
      eq(youtubeCache.cacheKey, cacheKey),
      sql`created_at > now() - interval '24 hours'`,
    ),
  });
  if (cached) return cached.results;
  
  const results = await searchYouTubeVideos({ query, courseCode });
  await db.insert(youtubeCache).values({ cacheKey, results });
  return results;
});
```

## 3. DMCA Takedown Process

### Automated Flagging
All uploaded content runs through the AI moderation pipeline. Content that appears to be commercially copyrighted course materials (textbooks, publisher PDFs) is flagged for human review before being made public.

### DMCA Notice Flow
1. Copyright holder submits takedown via `legal@studyscroll.dev`
2. Automated acknowledgement within 24 hours
3. Content temporarily hidden within 4 hours of valid notice
4. Uploader notified (email) with option to file counter-notice
5. If no counter-notice within 10 business days, content permanently removed
6. If counter-notice filed, content restored after 10 business days unless rights holder files suit

```typescript
// Takedown command
export const dmcaTakedown = command(async ({ postId, reason, reporterEmail }) => {
  // Hide immediately
  await db.update(posts).set({ isVisible: false, isFlagged: true })
    .where(eq(posts.id, postId));
  
  // Notify uploader
  await email.send({
    to: post.author.email,
    template: 'dmca-notice',
    data: { postId, reason },
  });
  
  // Log for legal record
  await db.insert(dmcaRequests).values({ postId, reason, reporterEmail, status: 'pending' });
});
```

## 4. Privacy Policy (GDPR / NDPR Compliance)

### Data Collected
| Data | Purpose | Retention |
|---|---|---|
| Email, username | Account creation, auth | Until account deletion |
| Course enrollments | Feed personalization | Until removed by user |
| Quiz answers, mastery scores | Adaptive algorithm | 2 years, then anonymized |
| Session/IP data | Security, fraud prevention | 90 days |
| Payment data | Billing (stored by Paystack/Stripe, not us) | As per provider |
| Uploaded notes/PDFs | OCR processing, content generation | Until deleted by user |

### Student Rights
- **Access**: Download all personal data via Settings > Account > Export Data
- **Deletion**: Delete account removes all personal data within 30 days
- **Portability**: Data export in JSON format
- **Opt-out**: AI content generation toggle, analytics opt-out

StudyScroll complies with Nigeria's NDPR by:
- Registering with NITDA as a data controller
- Providing a clear Privacy Policy in plain English
- Obtaining explicit consent for marketing emails
- Storing Nigerian user data on servers that comply with NDPR standards


```
for [COURSE CODE]. Verify against your official course materials."
```

### AI-Free Mode
Users can fully disable AI content via Settings > Content > AI-Generated Content toggle. This sets `users.aiEnabled = false`, and the feed algorithm filters out all `aiGenerated = true` posts.
